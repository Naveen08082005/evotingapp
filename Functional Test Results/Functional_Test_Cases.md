# Functional Test Cases: E-Voting Application

This document lists **25 comprehensive functional test cases** compiled for the Flutter + Supabase E-Voting Application. These cases cover authentication, administrative capabilities, student/voter modules, security guards, and real-time operations.

The spreadsheet version of these cases is available at: [functional_test_cases.xlsx](file:///d:/projects/evoting_app/Functional%20Test%20Results/functional_test_cases.xlsx).
